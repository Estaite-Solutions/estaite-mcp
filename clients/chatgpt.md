# ChatGPT

**Step 1** — In ChatGPT, go to **Settings → Apps**.

**Step 2** — Click **Advanced settings** and enable **Developer Mode**.

**Step 3** — Click **Create**.

**Step 4** — Fill in the details:

```
App name:        Estaite
MCP Server URL:  https://mcp.estaite.com?key=YOUR_API_KEY
Authentication:  None
```

**Step 5** — Click **Create** and tag **Estaite** in any chat to query rental market data.
