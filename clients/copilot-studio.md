# Microsoft Copilot Studio

**Step 1** — In your agent, click **Tools → Add a tool → Model Context Protocol**.

**Step 2** — Fill in the details:

```
Server name:     Estaite
Server desc:     US rental market intelligence
Server URL:      https://mcp.estaite.com
Authentication:  API Key
Type:            Query
Label:           key
```

**Step 3** — Click **Create**, then **Create a new connection**.

**Step 4** — In the **key** field, enter your API key.

**Step 5** — Click **Create**, then **Add and Configure**.
