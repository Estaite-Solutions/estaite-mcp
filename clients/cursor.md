# Cursor

**Step 1** — Go to **Cursor → Settings → Tools & Integrations → Add Custom MCP**.

**Step 2** — Add the configuration:

```json
{
  "mcpServers": {
    "estaite": {
      "url": "https://mcp.estaite.com?key=YOUR_API_KEY"
    }
  }
}
```

**Step 3** — Restart Cursor and verify the Estaite tools appear in your tools list.
