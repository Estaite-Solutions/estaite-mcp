# OpenAI Agent Builder

**Step 1** — Go to [OpenAI Agent Builder](https://platform.openai.com/agent-builder) and create a new flow.

**Step 2** — Click the agent node, then **Tools → Add MCP**.

**Step 3** — Enter the server URL with your API key included:

```
https://mcp.estaite.com?key=YOUR_API_KEY
```

**Step 4** — Set authentication to **None**.

**Step 5** — Click Connect, select the tools you want, then preview your agent.
